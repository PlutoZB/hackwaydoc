## 课程说明
机器学习简介，提供关于人工智能和不确定性推理的概率观点。主题包括：监督学习、集成方法和提升、神经网络、支持向量机、核方法、聚类和无监督学习、最大似然、图形模型、隐马尔可夫模型、推理方法和计算学习理论。学生应该熟悉多元微积分、线性代数、概率论和复杂性理论。学生将被要求用 Python 编写重要的程序。
