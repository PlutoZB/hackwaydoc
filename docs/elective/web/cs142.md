## 课程说明
用于构建交互式 Web 应用程序的概念和技术。浏览器端网络设施，例如 HTML、级联样式表、文档对象模型和 JavaScript 框架，以及服务器端技术，例如服务器端 JavaScript、会话和面向对象的数据库。Web 安全和应用程序可扩展性方面的问题。Web 应用程序部署的新模型。

先决条件： CS 107和 CS 108。
