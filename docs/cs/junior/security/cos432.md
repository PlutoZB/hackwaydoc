## 课程说明
如何保护计算系统、通信和用户。基本密码学；私人和经过身份验证的通信；软件安全；恶意软件；操作系统保护；网络安全; 网络安全；人身安全；加密货币和区块链；隐私和匿名；可用的安全性；安全经济学；安全伦理；法律和政策问题。
