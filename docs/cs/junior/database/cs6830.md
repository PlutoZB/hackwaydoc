## 课程说明
与数据库系统的工程和设计相关的主题，包括数据模型；数据库和模式设计；模式规范化和完整性约束；查询处理；查询优化和成本估算；交易；恢复; 并发控制；隔离和一致性；分布式、并行和异构数据库；自适应数据库；触发系统；发布-订阅系统；半结构化数据和 XML 查询。
