## 课程说明
本课程研究用于访问、搜索和管理信息的系统。我们详细研究了数据库系统用来管理结构化数据的方法。我们还考虑了半结构化和非结构化信息集合的组织和搜索，例如 Web 和数字图书馆，同时考虑了数据库概念和新模型的应用。正确性、性能和系统要求贯穿始终。一些特定主题：关系数据模型；XML 模型；索引；查询构造和评估；交易；搜索效率。
