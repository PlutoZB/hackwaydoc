# 普林斯顿 COS375 计算机体系结构




>**课程名称：** Computer Architecture  
**课程官网地址：**[2015年秋](https://www.cs.princeton.edu/courses/archive/fall15/cos375/)    
**先修课程：** 无  
**重要程度：** ※※※※※  
**课程评点：** 

## 课程说明
介绍计算机体系结构和组织。指令集设计；基本的处理器实现技术；绩效衡量；缓存和虚拟内存；流水线处理器设计；RISC架构；成本、性能和复杂性之间的设计权衡。
