## 课程说明
本课程介绍编程语言的理论、设计和实现。本课程涵盖的主题包括：编程语言的形式语义（操作、公理、指称和翻译）、类型系统、高阶函数和 lambda 演算、懒惰、延续、动态类型、单子、对象、模块、并发和沟通。
