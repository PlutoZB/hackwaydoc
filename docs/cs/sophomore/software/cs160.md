## 课程说明
CS 160 是人机交互 (HCI)领域的入门课程。在本课程中，学生将学习设计、制作原型和评估用户界面。与大多数课程不同，CS 160 不会专注于任何特定的算法技术集，相反，学生将学习以用户为中心的界面设计技术（例如，原型设计、上下文查询、启发式评估等）。今年夏天，我们将学习如何使用前端 Web 开发技术（HTML、CSS 和 JavaScript）和后端进行原型设计。
