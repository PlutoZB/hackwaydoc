# MIT 6.033 计算机系统工程

>**课程名称：** Computer System Engineering      
**课程官网地址：**[MIT 6.033课程官网](http://web.mit.edu/6.033/www/)

## 课程说明
6.033涵盖四个单元的技术内容：操作系统、网络、分布式系统和安全。讲座提供基本的技术概念；背诵向您展示了如何将这些概念应用于实际系统。在教程中，您将学习沟通技巧，以帮助您对系统进行推理，并描述和捍卫自己的系统。
