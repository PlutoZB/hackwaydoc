## 课程说明
CS 61介绍了计算机系统编程的基础知识。主题包括 C、C++ 和汇编语言编程、性能分析和改进策略、内存管理、缓存、并发、线程和同步。

CS 61 将帮助您培养为现实世界编写程序的技能，在现实世界中性能和健壮性非常重要。它还将为您准备更高级的 CS 课程，包括操作系统、编译器和编程语言、体系结构和图形。我们希望它充满乐趣和挑战性。
